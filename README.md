# memorizer_words

It's a desktop app to help language learners memorizing words that they found. (Windows and Linux)

## How to build 

Clone the repository.

```bash
git clone https://github.com/ChristianFLozano/memorizer_words.git
```

Now, you must have QT CREATOR (Community) installed.

- Open QT CREATOR.
- Click on File -> Open File or Project,
- Search for the CMakeLists.txt and open it.
- Configure the project using DESKTOP QT 6 kit.
- Press RUN.
